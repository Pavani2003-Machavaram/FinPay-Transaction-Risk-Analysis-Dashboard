# 💳 FinPay Transaction & Risk Analysis Dashboard

## Overview

The **FinPay Transaction & Risk Analysis Dashboard** is an interactive Power BI project designed to analyze digital wallet transactions, customer behavior, and fraud risk. The dashboard provides meaningful insights into transaction performance, payment methods, and risk patterns through dynamic visualizations, KPIs, and interactive filters, enabling data-driven business decisions.

## Project Objectives

* Analyze digital wallet transaction performance.
* Monitor transaction trends over time.
* Identify high-risk and potentially fraudulent transactions.
* Compare customer transaction behavior.
* Analyze payment methods and transaction status.
* Enable interactive data exploration using slicers and filters.

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Dashboard Components

### KPI Cards

The dashboard provides key business metrics including:

* Total Transactions
* Total Transaction Amount
* Average Transaction Amount
* Successful Transactions
* Failed Transactions
* Fraud Risk Transactions

### Interactive Filters

Users can filter data by:

* Transaction Type
* Payment Method
* Transaction Status
* Customer
* Date/Month

## Visualizations

### Transaction Trend Analysis

Displays transaction volume and amount over time to identify business trends.

### Payment Method Analysis

Compares transactions across different payment methods.

### Fraud Risk Analysis

Highlights high-risk transactions and fraud patterns.

### Customer Transaction Analysis

Shows customer-wise transaction activity and spending behavior.

### Transaction Status Analysis

Compares successful, failed, and pending transactions.
